Syngenic RM-Silencing of Genetic Constructs

SyToGen performs syngenic DNA-based RM-silencing of genetic constructs, allowing bacteriologists to overcome restriction-modification barriers without altering the encoded protein sequence.

SyToGen Input

  • Motif-mapped GenBank file from MotifFinder (.gb / .gbk / .gbff)
  • Strain-specific codon usage table from CodonBias (.csv)
  • Restriction motif table (.tsv / .txt / .csv)

SyToGen Output

  • RM-silent construct (FASTA)
  • Annotated result (GenBank)
  • Motifs used in this run

About SyToGen

SyToToGen enables systematic RM-silencing of genetic constructs. It automates motif screening, codon-aware redesign, and construct reconstruction, reducing the need for specialized expertise in sequence engineering.

Manual RM-silencing does not scale well across strain collections. SyToGen enables broader redesign strategies, allowing constructs to function across many strains.

What does SyToGen do?

  • Analyzes motif occurrences across the construct
  • Identifies valid synonymous single-base edits
  • Preserves strain-specific codon usage preferences
  • Prevents introduction of new recognition sites
  • Generates ranked redesign options
  • Outputs a complete RM-silent construct
