SyToGen performs syngenic DNA-based RM-silencing of genetic constructs.

SyToGen requires the following files:
The MotifFinder-annotated GenBank file to show SyToGen which regions to target. The strain-specific codon usage table from CodonBias to guide codon optimization.

SyToGen Input:

  • Motif-mapped sequences from MotifFinder (.gbk)
  • Codon-usage table from CodonBias (.csv)
  • Editing parameters (optional constraints and rules)

SyToGen Output:

  • RM-silent construct (GenBank + FASTA)
  • Detailed edit report
  • Alternative candidate sequences
  • Graphical summaries and logs

About SyToGen

SyToGen enables systematic RM-silencing of genetic constructs, allowing bacteriologists to overcome restriction-modification barriers.

It automates motif screening, codon-aware redesign, and construct reconstruction, reducing the need for specialized expertise in sequence engineering.

Manual RM-silencing does not scale well across strain collections. SyToGen enables broader redesign strategies, allowing constructs to function across many strains.

What does SyToGen do?

  • Analyzes motif occurrences
  • Identifies valid synonymous edits
  • Preserves codon usage preferences
  • Prevents new motif introduction
  • Generates ranked redesign options
  • Outputs a complete RM-silent construct
