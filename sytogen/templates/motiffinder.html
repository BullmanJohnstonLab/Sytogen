MotifFinder

MotifFinder allows users to map RM motifs across a provided genetic construct, or a set of constructs, to identify every position in the sequence that would be recognized and cleaved by the target strain's restriction enzymes.


By locating these sites precisely, MotifFinder determines where the syngenic DNA RM-silencing strategy must introduce sequence edits to improve transformation efficiency.

About MotifFinder

MotifFinder parses a user-supplied GenBank or FASTA file, extracts the annotated sequence, and screens it for all RM target motifs defined in the MyMotifs module. For every motif occurrence, the module records its position, strand orientation, sequence context, and overlap with annotated features.

In addition to mapping motifs on a single construct, MotifFinder can be used to screen multiple plasmids or replicons to compare their motif burdens. Outputs are a GFF3 annotation file and a TSV summary table, both packaged in a ZIP archive for downstream use by the SyToGen redesign module.
